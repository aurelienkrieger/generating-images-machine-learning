# Prompt engineering

## Key topics

- prompt
- negative prompt
- keywords weight
- keyword blend
- prompt generators & boilerplates
- reverse prompting
- regional prompting

## Learning points

- describe the subject in a natural language and in the most detailed way
- add keywords following the structure:
    - Medium
    - Style
    - Artist / Inspiration
    - Website	Resolution
    - Details
    - Color
    - Lighting
- weight keywords
- use negative prompts
- use keyword blending
- know how to play with parameters
- use prompt boilerplates for specific styles
- use reverse prompting

## Parameters

- Classifier Free Guidance (CFG) scale
- Sampling steps
- Image size
- Sampling method
- Seed
- Batch size
